# Your personal search engine

![Home](https://sleep.deta.dev/cdn/surfer.png)
> Theme: Ocean Blue

## Features
- Cross space app integrations
- Site/Image Bookmarks
- History (Toggable)
- Moblie supported
- Themes (12)
- Safesearch
- Surfcuts

## Surfcuts
- `>surf:home`
- `>surf:history`
- `>surf:settings`
- `>surf:bookmarks`
- `>surf:image query here`


## Integrations

- Use surfer's image search directly in your blackhole app
